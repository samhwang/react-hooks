# useEffectOnce

A hook that runs the effect only once.

## Usage

```typescript
useEffectOnce(callback);
```

## Parameter

| Parameter | Description                                      | Type     | Default    |
| --------- | ------------------------------------------------ | -------- | ---------- |
| callback  | The callback function to execute on first render | function | `() => {}` |

## Return

This hook does not return anything
